Documentation

This app was developed with Electron and NodeJS as its backend. You can toggle the developer tools by Ctrl+I (if you have installed from source).


Getting Started

  • Install Arduino IDE and the required drivers
  • Note the COM Port assigned to your microcontroller in device manager.

Windows

COM ports in Windows follow the format COM[0-9] and are not case sensitive.


Ubuntu and Debian

Linux usually assigns /dev/ttyACM0 for microcontrollers.

You can give R/W access (chmod 777) to the current user by -

sudo usermod -a -G dialout <user_name>
sudo chmod a+rw /dev/ttyACM0
